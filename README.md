# Real-Time Chat & Video Application

A full-stack real-time communication platform that supports group-based messaging, image sharing, and peer-to-peer video calls.

## Features
- Real-time messaging using WebSockets (Socket.io)
- Peer-to-peer video calls and screen sharing (PeerJS)
- Group and channel-based communication system
- User authentication and role-based permissions (admin / super admin)
- Image upload support for profiles and groups
- Dynamic UI updates without page reload

## Tech Stack
- **Frontend:** Angular, TypeScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Real-time:** Socket.io, PeerJS

## Key Highlights
- Designed a scalable chat system with separation between frontend and backend services
- Implemented real-time communication using WebSockets for instant updates
- Integrated PeerJS for low-latency video and screen sharing
- Built RESTful APIs for managing users, groups, and messages
